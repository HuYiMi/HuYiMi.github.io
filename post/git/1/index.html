git安装及使用

发表于2021-12-20|更新于2021-12-14|git
|字数总计:288|阅读时长:1分钟|阅读量:|评论数:

一.安装环境

1.本机系统：Windows 10(64位)

2.git：2.34.1(64位)

二.下载路径 https://git-scm.com/downloads

三.开始安装

1.选定自己下载的版本

2.下载成功后，双击”Git-2.34.1-64.exe”,开始安装git

3.点击【Next】按钮

4.点击【Next】按钮

5.点击【Install】按钮

6.点击【Finish】按钮

7.查看git是否安装成功

四.git使用

1.设置用户名及邮件

2.将本地的master分支推送到origin主机

3.从远程获取代码并合并本地的版本

4.允许不相关的历史记录

5.把github项目克隆到本地文件夹中

6.如果报OpenSSl SSL_read:Connection was reset,errno 1054

7.添加ssh key

8.检测ssh是否添加成功

9.把本地项目上传到GitHub中
